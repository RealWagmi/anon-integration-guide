# debridge

deBridge is DeFi's internet of liquidity, enabling real-time movement of assets and information across the DeFi landscape

## Supported Networks

- ethereum
- optimism
- bsc
- polygon
- sonic
- fantom
- metis
- base
- avalanche
- arbitrum
- gnosis
