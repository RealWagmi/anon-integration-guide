# beefy

Beefy is a Decentralized, Multichain Yield Optimizer that allows its users to earn compound interest on their crypto holdings. Beefy earns you the highest APYs with safety and efficiency in mind.

## Supported Networks

- ETHEREUM
- BSC
- BASE
- AVALANCHE
- ARBITRUM
- SONIC
